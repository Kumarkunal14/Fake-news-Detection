# 📰 Fake News Detection Web App

## 🚀 Overview
This project is a Machine Learning-based web application that detects whether a news article is **Real or Fake** using Natural Language Processing (NLP).

---

## 🧠 Features
- Detects fake vs real news instantly
- Uses TF-IDF vectorization
- Machine Learning model (Passive Aggressive Classifier)
- Confidence score display
- Web interface built using Flask

---

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

---

## ⚡ How It Works
1. User enters news text
2. Text is processed using TF-IDF
3. Model predicts Real or Fake
4. Result displayed with confidence score

---

## 📊 Model Details
- Algorithm: Passive Aggressive Classifier
- Dataset: Fake & Real News Dataset
- Accuracy: ~85-90%
